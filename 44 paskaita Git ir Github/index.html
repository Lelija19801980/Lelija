Git ir Github

Git kodo versijamimo programa

Github - versijuoto kodo talpykla internete

Git komandos:

  • git init - sukurti repozitoriją (pgit hidden skyltyje)
  • git status - kas yra dabar
  • git add - prideda nurodytus failus prie comit paruostus failus
  • git rm --cached || restore --staged - išima nurodytus failus ir stage zonos
  • git comit -m "zinute apie comita" sukuria kodo versija ir stage failų ir prideda zinute apie pakitimus(zinute turi buti trumpa ir aiski)
  • restore - sugrazina nurodyta ar nurodytus failus i pradine busena, jei tie failai nebuvo edinti i stage zona
  • git push - nusiuncia lokaliai su comitintus pakitimus i isorine repozitus
  • git pull - atsiuncia isorines repozitorijos comitus pakitimus i lokalia repozitorija
  • git log - isveda comitu sarasa
  • git clone - atsiusti visa projekto koda, klonuoti i dabartini aplankala
  • git checkout commitID|branchName - pereianama i nurodyta kodo versija arba kita saka. Tada matosi kita saka.
  • git branch naujosSakosPavadinimas - sukurti nauja saka
  • ....
